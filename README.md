# PrimaThinkIMS - Intern Management System

A comprehensive platform for managing intern onboarding, task assignments, and progress tracking.

## Features

- Structured intern onboarding with step-by-step registration
- Task management with sequential progression
- Admin dashboard for oversight and task assignment
- Role-based access control

## Technologies

- React + TypeScript
- Vite
- Tailwind CSS
- shadcn/ui

## Getting Started

```sh
# Install dependencies
npm install

# Start development server
npm run dev
```
